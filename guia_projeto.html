Avaliação

As 5 Métricas RAGAS

Faithfulness
Answer ↔ Context
O LLM usou o contexto recuperado para gerar a resposta? Alto = sim. Baixo = inventou.
Answer Correctness
Answer ↔ Ground Truth
A resposta está factualmente correta em relação ao ground truth esperado?
Answer Relevance
Answer ↔ Question
A resposta gerada é relevante e diretamente relacionada à pergunta feita?
Context Precision
Context ↔ Q + GT
O sistema de retrieval recuperou chunks realmente úteis, sem ruído?
Context Recall
Context ↔ Ground Truth
O contexto recuperado contém toda a informação necessária para responder?
💡
Insight: Faithfulness baixa + Correctness alta
Se o LLM responde certo mas com faithfulness baixa, significa que ele usou seu próprio conhecimento interno — não o contexto recuperado. Em aplicações que exigem rastreabilidade e confiabilidade (como sistemas corporativos ou médicos), isso é um problema: você não pode auditar de onde veio a informação.
Início rápido

Como executar

1
Instalar dependências
pip install -r requirements.txt
Python 3.10+ recomendado
2
Configurar chave da OpenAI
export OPENAI_API_KEY="sk-..."
Ou crie um arquivo .env com OPENAI_API_KEY=sk-...
3
Adicionar seus documentos (opcional)
cp meus_docs/*.txt data/sample_docs/
O projeto já inclui um documento de exemplo sobre história da IA
4
Gerar o Golden Dataset
python src/golden_dataset.py
Gera outputs/golden_dataset.csv com perguntas + ground truth
5
Avaliar o sistema RAG
python src/evaluate.py
Gera outputs/evaluation_scores.csv com os 5 scores RAGAS
